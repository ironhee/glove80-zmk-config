**Tap**: Enables `NUMB` layer

```text
┌─────┐
│→NUMB│
└─────┘
```

**Tap**: disables `NUMB` layer

```text
┌─────┐
│←NUMB│
└─────┘
```

**Hold**: Momentary enables `NUMB` layer

```text
Layers - DEFT
┌─────┐
│     │
└▀NUMB┘
```

`NUMB` layer is enabled momentarily while the key is pressed

```text
Layers - NUMB
┌─────┐

└▀▀▀▀▀┘
```
