# Glove80 zmk keymap

<details>
<summary>Guide</summary>


**Tap**: `A`

```text
╭─────╮
│ A   │
╰─────╯
```

**Tap**: Enables `NUMB` layer

```text
╭─────╮
│→NUMB│
╰─────╯
```

**Tap**: `A`, **Hold**: Momentary Enables `NUMB` layer

```text
╭─────╮
│ A   │
╰■NUMB╯
```

**Tap**: `A`, **Hold**: `Shift`

```text
╭─────╮
│ A   │
╰─Sft─╯
```

**Tap**: `Shift`,  **Double Tab**: `Caps Word`

```text
╭──Caps
│ Sft │
╰─────╯
```

**Tap**: `/`, **Shift Tap**: `\`

```text
╭─────╮
│ / \ │
╰─────╯
```

**Pressed**

```text
╭─────╮
│     │
╰▀▀▀▀▀╯
```
